# 🚩 README
Flag Quiz - Backend Practice This project is an interactive web application designed to test users' geographical knowledge. Developed as part of my training in Angela Yu's Full Stack Web Development bootcamp. 🚀 Technologies Used Node.js: Runtime environment for the backend. Express: Framework for route and server management. PostgreSQL: Relational database for storing countries and flags. EJS (Embedded JavaScript templates): Template engine for dynamically rendering views. CSS: Custom styles (TrazoDigital Design). 🛠️ Features Real-time connection to a PostgreSQL database. Cumulative scoring system. Server-side response validation. Responsive interface with a world flags theme. 📦 Installation and Configuration Clone the repository: git clone [https://github.com/rodolfomartinezmiranda-png/Postgres-Flag-Quiz.git](https://github.com/rodolfomartinezmiranda-png/Postgres-Flag-Quiz.git) Install the dependencies: npm install Configure your PostgreSQL database with the scripts included in the introduction folder. Create an .env file with your database credentials.
